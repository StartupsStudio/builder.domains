## Behavioral Economics of Free

### The Sunk Cost Fallacy Eliminated

**With paid domains:**

```python
class BuilderPsychology:
    def evaluate_idea(self, idea):
        if self.domains_cost > 0:
            # Pre-commitment anxiety
            questions = [
                "Is this worth $15?",
                "Will I actually finish this?",
                "What if it fails?",
                "Can I afford to waste $15?"
            ]

            # Decision paralysis
            if self.confidence < 0.8:
                return "DON'T_START"  # Idea dies

        return "START"

    def abandon_project(self, project):
        if project.domain_cost > 0:
            # Sunk cost fallacy
            guilt = project.domain_cost / self.hourly_rate

            # Keep working to "justify" the $15
            # Even if project has no future
            return self.continue_project(project)
```

**With free domains:**

```python
class BuilderPsychology:
    def evaluate_idea(self, idea):
        # No financial barrier
        # No pre-commitment needed
        return "START"  # Just try it

    def abandon_project(self, project):
        # No sunk cost
        # No guilt
        # Easy to pivot
        return self.move_to_next_idea()
```

  calculateUpgradeProbability(domain: Domain) {
    const signals = {
      traffic: domain.monthlyVisitors > 1000 ? 0.3 : 0,
      revenue: domain.hasRevenue ? 0.4 : 0,
      team: domain.teamSize > 1 ? 0.2 : 0,
